Introduction

This page is made to fulfill the web mapping final task which consits of two parts.

Course accompanying website

  • The main aim of this part is creating a course accompanying website with predefined content items.
  • We should create a landing page and at least 4 sub pages with the following content: Comparison of online mapping services, Google Maps API, OpenStreetMap, Usability
  • The website must contain some information about its author.
  • We should create an individual website style with consistent navigation and design. The website should be aesthetic and user-friendly
  • All pages must share one external css file.
  • At least your landing page should validate without errors on Validator W3.
  • The website must be publicly available on a server.

Web mapping project

  • The main aim of this part is working on a web mapping project and presenting the results on the website
  • There are several options for the mapping project
    • Practical project (individual or groups of 2-3 students)
    • Theoretical project (individual)
    • Participation in one of our research projects
  • I have chosen to do practical project so I had to chose my topic, create concept, find or collect the data, build the application and integrate it in your website, and write a technical report
  • My topic is My skiing trip to Pra Loup (France) and you can find the link in the navigation bar as My project.
